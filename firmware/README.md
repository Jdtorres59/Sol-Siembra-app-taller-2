# Firmware

Este directorio almacenará el código y la documentación del firmware para la ESP32 y demás dispositivos del sistema.

## Propuesta de estructura

- `esp32/`: proyecto de firmware principal.
  - `src/`: archivos fuente (se agregarán más adelante).
  - `include/`: encabezados, definiciones de pines, constantes.
  - `lib/`: bibliotecas locales reutilizables.

Puedes adaptar esta estructura a PlatformIO, Arduino IDE u otra herramienta según se necesite.
